# Changelog

所有对 Webtor 项目的显著更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 添加

- 初始项目设置。
- 基本的终端模拟功能。
- 多标签支持。
- 深色/浅色主题切换。
- 保存和加载服务器配置。

## [0.0.1] - 2024-09-21

### 添加

- 创建基本的项目结构。
- 实现基础的 SSH 连接功能。
- 添加响应式设计。
- 集成 xterm.js 作为终端模拟器。
- 实现多标签管理。
- 添加主题切换功能。
- 添加服务器配置保存和加载功能。

### 变更

- 优化用户界面布局。
- 改进错误处理和用户反馈。

### 修复

- 修复了终端大小调整问题。
- 解决了在某些情况下连接断开的问题。
