# Webtor

Webtor 是一款轻量级的基于浏览器的远程服务器管理工具。它提供了一个直观的 Web 界面，允许用户通过浏览器连接和管理远程服务器。

## 特性

- 基于浏览器的终端模拟器。
- 支持多个终端标签。
- 保存和管理多个服务器配置。
- 自适应深色/浅色主题。
- 响应式设计，支持移动设备。

## 技术栈

- 前端：Vue 3, Tailwind CSS, Xterm.js
- 后端：Node.js, Express, WebSocket
- 构建工具：Vite

## 安装

1. 克隆仓库：
   ```bash
   git clone https://github.com/sscaifesu/webtor.git
   cd webtor

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 访问 `http://localhost:5173` 查看应用。
